## SPL Token CLI

Command-line tools for managing SPL tokens on Solana.

### Token Management

- [Create Token](/cli/create-token) - Create a new SPL token
- [Create Account](/cli/create-account) - Create a token account
- [Mint Tokens](/cli/mint-tokens) - Mint tokens to an account
- [Burn Tokens](/cli/burn-tokens) - Burn tokens from an account

### Account Operations

- [Close Token Account](/cli/close-token-account) - Close a token account
- [Get Supply](/cli/get-supply) - Get token supply info
- [Display Information](/cli/display-information) - Display token info

### Authority Management

- [Disable Freeze](/cli/disable-freeze) - Disable freeze authority
- [Disable Mint](/cli/disable-mint) - Disable mint authority
